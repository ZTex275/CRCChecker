# Мультиязычность в CRC Checker

## Обзор
Приложение поддерживает два языка: русский и английский. Переключение языка осуществляется с помощью кнопок EN/RU в верхней части интерфейса.

## Реализация

### 1. LocalizationManager.cs
- **Класс**: `LocalizationManager` (singleton)
- **Перечисление**: `SupportedLanguage` (English, Russian)
- **Свойства**: Все строки интерфейса как свойства с автоматическим переключением
- **События**: `LanguageChanged` для обновления UI при смене языка

### 2. MainWindow.xaml
- Добавлены кнопки переключения языка (EN/RU)
- Все текстовые элементы привязаны к `LocalizationManager`
- Стили для активной/неактивной кнопки языка

### 3. MainWindow.xaml.cs
- Обработчики кнопок языка
- Автоматическое обновление UI при смене языка
- Поддержка локализованных сообщений об ошибках

## Использование
1. Запустите приложение
2. Нажмите кнопку EN для английского языка или RU для русского
3. Весь интерфейс мгновенно переключится на выбранный язык
4. Выбор языка сохраняется на время работы приложения

## Поддерживаемые элементы
- Заголовок окна и приложения
- Кнопка "Обзор..." / "Browse..."
- Все метки полей (Имя файла, Размер файла и т.д.)
- Диалоговые окна выбора файла
- Сообщения об ошибках
